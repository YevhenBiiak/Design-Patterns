# Design-Patterns

The result of my learning of design patterns. 

A simple application with a list of patterns grouped by categories.
<br>All the images as well as the design of the application are designed in Sketch. 
<br>Pattern structure images are taken from: [refactoring.guru](https://refactoring.guru/design-patterns/catalog). 

Each pattern has a brief description, popularity and complexity indicators for swift developers, and code example. 
<br>Code example is downloaded from this git repository and has an implementation of a specific situation.

Text in code examples is colored with NSAttributedString and Regex
